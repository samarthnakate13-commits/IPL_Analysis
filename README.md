# IPL_Analysis
Ball-by-ball analysis of IPL matches (2008-2026) using pandas and matplotlib — top scorers, wicket takers, toss impact, and scoring trends.
## Dataset
This project uses the [IPL Ball-by-Ball Dataset (2008-2025)](https://www.kaggle.com/datasets/chaitu20/ipl-dataset2008-2025) from Kaggle. Due to GitHub's file size limit, the CSV is not included in this repo — download it from the link above and place it in the project folder as `IPL.csv` to run the notebook.
## Questions Explored
1. Who are the top 10 run scorers across all IPL seasons?
2. Who are the top 10 wicket takers?
3. Which team has the highest win percentage?
4. Does winning the toss actually help you win the match?
5. Which season had the highest total score?

## Key Findings
- Virat Kohli leads all-time scoring with 8899 runs, ~1700 ahead of Rohit Sharma.
- Yuzvendra Chahal leads wicket-taking charts, ahead of higher-profile pacers like Bumrah.
- Mumbai Indians have the most wins, but win percentage shows a tighter race among top teams.
- Toss win translates to match win only ~52% of the time — barely better than a coin flip.
- 2024 had the highest total runs scored across all seasons, reflecting a shift toward batting-friendly conditions.
