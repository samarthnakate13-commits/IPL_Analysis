# IPL_Analysis
Ball-by-ball analysis of IPL matches (2008-2026) using pandas and matplotlib — top scorers, wicket takers, toss impact, and scoring trends.
## Dataset
This project uses the [IPL Ball-by-Ball Dataset (2008-2025)](https://www.kaggle.com/datasets/chaitu20/ipl-dataset2008-2025) from Kaggle. Due to GitHub's file size limit, the CSV is not included in this repo — download it from the link above and place it in the project folder as `IPL.csv` to run the notebook.
